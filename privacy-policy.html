Privacy Policy

Introduction

SatsSpark is a decentralized wallet extension that supports Spark Network, Bitcoin mainnet, and Lightning Network. We take your privacy and data security very seriously.

Data Collection

  • Private Keys and Seed Phrases: Stored only on your local device, never uploaded to our servers
  • Wallet Addresses: Used only for blockchain transactions, not linked to personal identity
  • Transaction Records: Retrieved from public blockchain networks, no private information collected

Data Usage

  • Browser local storage is used to save wallet settings and preferences on your device only
  • Network requests are used only to retrieve blockchain data and execute transactions
  • No data is sent to our servers - all operations are client-side only
  • We do not collect or share any personally identifiable information

Third-Party Services

  • Spark Network API: For LRC-20 token interactions and balance queries
  • Mempool.space: For Bitcoin network status and transaction data
  • CoinGecko API: For real-time cryptocurrency price information
  • Lightning Network Nodes: For Lightning Network payment channels
  • We do not share data with any advertising, analytics, or data collection services

WebAssembly Usage

  • We use WebAssembly for high-performance cryptographic computations
  • All computations are executed locally, data never leaves your device
  • Used only for Bitcoin and Lightning Network cryptographic operations

Permissions Explanation

  • Network Access: Only connects to necessary blockchain API services including Spark Network, Bitcoin mempool, and price data APIs
  • Local Storage: Uses browser's local storage (not Chrome extension storage) to save wallet settings and cache data locally
  • No Cross-Tab Access: Extension operates independently without accessing other browser tabs or web pages
Key Security Promise: Your private keys and seed phrases are encrypted and stored locally on your device only. We have no access to your funds or private information.

Data Security

  • All sensitive data is encrypted and stored locally
  • We use industry-standard encryption algorithms
  • Private keys and seed phrases are never transmitted to third parties

User Rights

  • You can delete wallet data at any time
  • You have complete control over your private keys and assets
  • We cannot access or recover your wallet data

Policy Updates

We may periodically update this privacy policy. Major changes will be communicated to users through extension updates.

Contact Us

For privacy-related questions, please contact:

  • Email: admin@satsspark.xyz
  • Website: https://satsspark.xyz

隐私政策

简介

SatsSpark是一个支持Spark网络、比特币主网和闪电网络的去中心化钱包扩展。我们非常重视您的隐私和数据安全。

数据收集

  • 私钥和助记词: 仅存储在您的本地设备上，永不上传到我们的服务器
  • 钱包地址: 仅用于区块链交易，不与个人身份关联
  • 交易记录: 从公开区块链网络获取，不收集私人信息

数据使用

  • 浏览器本地存储仅用于在您的设备上保存钱包设置和偏好
  • 网络请求仅用于获取区块链数据和执行交易
  • 不向我们的服务器发送任何数据 - 所有操作均在客户端完成
  • 不收集或分享任何个人身份信息

第三方服务

  • Spark Network API: 用于LRC-20代币交互和余额查询
  • Mempool.space: 用于比特币网络状态和交易数据
  • CoinGecko API: 用于获取实时币价信息
  • Lightning Network节点: 用于闪电网络支付通道
  • 不与任何广告、分析或数据收集服务共享数据

WebAssembly 使用

  • 使用WebAssembly进行高性能密码学计算
  • 所有计算在本地执行，数据不离开您的设备
  • 仅用于比特币和闪电网络的加密操作

权限说明

  • 网络访问: 仅连接必要的区块链API服务，包括Spark网络、比特币内存池和价格数据API
  • 本地存储: 使用浏览器本地存储（非Chrome扩展存储）在本地保存钱包设置和缓存数据
  • 无标签页访问: 扩展独立运行，不访问其他浏览器标签页或网页内容
核心安全承诺： 您的私钥和助记词经过加密后仅存储在您的设备本地。我们无法访问您的资金或私人信息。

数据安全

  • 所有敏感数据均在本地加密存储
  • 使用行业标准的加密算法
  • 不向任何第三方传输私钥或助记词

用户权利

  • 您可以随时删除钱包数据
  • 您完全控制自己的私钥和资产
  • 我们无法访问或恢复您的钱包数据
